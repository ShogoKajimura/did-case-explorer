# Public Codebook 2026-03-27

This file documents the public-facing fields released through the companion
explorer snapshot.

## Core bibliographic fields

- `id`
- `authors`
- `year`
- `title`
- `journal`
- `doi`
- `pmid`
- `language`
- `languageGroup`
- `country`

## Structure fields

- `recordType`
- `recordTypeCode`
- `caseShape`
- `caseCount`
- `caseCountNumeric`

## Access fields

- `accessCode`
- `accessLabel`
- `publicLinkUrl`

These indicate what is publicly exposed.

They do **not** imply that copyrighted full text is redistributed through the
site.
