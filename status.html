Polling behavior

Pending and processing jobs are polled automatically. Once OCR completes, the page stops polling and enables secure download actions.

Security boundary

The token lives in the private link itself and is sent to the restricted backend only when this page requests status or downloads.

Status tracking is private and token-gated. Public metadata release remains a separate editorial decision.
